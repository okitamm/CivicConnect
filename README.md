**Simpler Access to Government Services, For Every Filipino.**

## 📖 Overview
CivicConnect is a multi-tenant government appointment platform built for the Philippines. It serves as a unified system to connect Filipino citizens with various government agencies. The platform ensures data privacy by allowing each government office to operate in an isolated environment, while simultaneously giving system administrators full platform-level visibility. 

## ✨ Key Features
* **Real-Time Appointment Booking:** Citizens can browse available services, select time slots, and receive instant confirmation with live slot availability updates.
* **Multi-Tenant Isolation:** Each government office operates in a fully isolated environment to enforce strict data sovereignty.
* **Staff Operations Dashboard:** Branch staff utilize a dedicated console to view today's queue, review citizen details, and execute one-click status updates like "Complete" or "No-Show".
* **Superadmin Control Panel:** The platform owner can register new government agencies, generate isolated admin credentials, and monitor system health.
* **Slot & Queue Management:** Office admins configure available time slots, daily slot capacities, and estimated service durations.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, and Vanilla JavaScript.
* **Backend & Database:** Firebase Authentication and Firebase Firestore.
* **SDK Versions:** Firebase App, Auth, and Firestore SDKs version `10.8.1` (compat).
* **Data Visualization:** Chart.js is implemented for real-time traffic analytics.
